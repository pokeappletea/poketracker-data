Setup Instructions
1. Create a new GitHub repository (can be private) — e.g. yourname/poketracker-data
2. Go to github.com/settings/tokens/new ↗, create a classic token with repo scope checked.
3. Paste the token, repo name, branch (main), and file path above, then click Save Config.
4. Click Push to GitHub to upload your current data. From any other device, open the same HTML file and click Pull from GitHub to load it.
⚠ Your token is stored in localStorage only — it never leaves your browser except when making GitHub API calls directly. Never share your token.
